Representation of real numbers.

Real numbers can be thought of, without loss of generality, as the set of integers union the set of rationals union the set of irrationals. The only primitive data type native to Rust that is able to represent all three of these classes is f64 (not counting the impossibility of completely describing irrational numbers as decimals), which is why it’s used as the struct’s internal representation of real values.
